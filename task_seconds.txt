На вход программе подаются два целых числа:
1. Число n — текущие секунды с начала суток.
2. Число k — секунды, которые нужно прибавить.

Программа должна посчитать, какое время (часы, минуты и секунды) 
покажут часы после того, как пройдёт k секунд. 
Числа n и k могут быть очень большими.

Входные данные: 
Два целых числа n >= 0 и k >= 0 (вводятся по очереди).

Выходные данные: 
Три целых числа через пробел — часы (0-23), минуты (0-59) и секунды (0-59).

Примеры:
Вводим 4000 и 10. Получаем: 1 6 50
Вводим 0 и 86400. Получаем: 0 0 0
